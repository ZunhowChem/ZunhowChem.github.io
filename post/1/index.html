GROMACS建模常用命令

   由于近期在构建分子动力学模型时经常翻教程查找命令从而耽误不少时间，故将建模常用命令总结与此，也会更新学会的新的命令。

   gmx editconf : 此命令是将其他化学结构的格式转化为GROMACS可识别的.gro，.g96，.pdb。
   gmx editconf  [-f [<.gro/.g96/...>]] [-n [<.ndx>]] [-o [<.gro/.g96/...>]]  [-box <vector>] [-center <vector>]
   [-f [<.gro/.g96/...>]] 结构文件：gro g96 pdb tpr
   [-n [<.ndx>]] （可选）：索引文件
   [-o [<.gro/.g96/...>]]（可选）：输出结构文件gro g96 pdb
   [-box <vector>]（0 0 0）：盒子边长(a,b,c)
   [-center <vector>]（0 0 0）：将几何中心移动到(x,y,z)

    gmx insert-molecules：此命令用于在已有的-ci结构中插入指定的-f系统中的空白空间中，或者-box给定一个空盒，再放入-ci结构。
   gmx insert-molecules [-f [<.gro/.g96/...>]] [-ci [<.gro/.g96/...>]]  [-n [<.ndx>]] [-o [<.gro/.g96/...>]]  [-nmol <int>] [-box <vector>]
   [-f [<.gro/.g96/...>]] 要插入的现有结构：gro g96 pdb tpr
   [-ci [<.gro/.g96/...>]]插入的结构：gro g96 pdb tpr
   [-n [<.ndx>]] （可选）：现有的索引文件
   [-o [<.gro/.g96/...>]]（可选）：输出结构文件gro g96 pdb
   [-nmol <int>]：需要插入的分子数
   [-box <vector>]（0 0 0）：盒子边长（nm）

   gmx solvate能解决2个问题：
   （1）创造一个溶剂盒子。尤其是-cs和-box，或者使用结构文件通过-cs和-cp。
   （2）解决溶剂配置。例如在溶剂中运动的蛋白质，指定-cp（溶质）和-cs（溶剂）。除非设置了-box，否则将使用在溶质坐标文件（-cp）中指定的盒子。
   gmx solvate [-cp [<.gro/.g96/...>]] [-cs [<.gro/.g96/...>]] [-p [<.top>]] [-o [<.gro/.g96/...>]] [-box <vector>]
   [-cp [<.gro/.g96/...>]] 结构文件（溶质）：gro g96 pdb tpr
   [-cs [<.gro/.g96/...>]] 结构文件（溶剂）：gro g96 pdb tpr
   [-p [<.top>]] 拓扑文件：top
   [-o [<.gro/.g96/...>]]（可选）：输出结构文件gro g96 pdb
   [-box <vector>] ：盒子边长（nm）

   gmx genion：使用单原子离子随机取代溶剂分子。
   gmx genion [-s [<.tpr>]] [-n [<.ndx>]] [-p [<.top>]] [-o [<.gro/.g96/...>]] [-np <int>] [-pname <string>] [-pq <int>] [-nn <int>] [-nname <string>] [-nq <int>]  [-[no]neutral]
   [-s [<.tpr>]] ：合理的xdr运行输入文件
   [-n [<.ndx>]] （可选）：现有的索引文件
   [-p [<.top>]] 拓扑文件：top
   [-o [<.gro/.g96/...>]] ：输出结构文件gro g96 pdb
   [-np <int>]（0）：阳离子个数
   [-pname <string>]（NA）：阳离子名称
   [-pq <int>]（1）：阳离子电荷数
   [-nn <int>]（0）：阴离子个数
   [-nname <string>]（CL）：阴离子名称
   [-nq <int>]（-1）：阴离子电荷数
    [-[no]neutral] ：加入适量的单原子离子使系统呈电中性
