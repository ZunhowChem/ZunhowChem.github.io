CGenFF产生小分子CHARMM力场拓扑文件

   分子动力学模拟中力场的质量决定了计算结果是否合理，本文通过介绍CGenFF工具生成小分子的拓扑文件。

需要准备的工具

   **CGenFF主页**：https://cgenff.umaryland.edu/ 需要注册用户，用户一周能上传150个分子结构进行拓扑文件的产生，分子内最大原子数不超过384。需要准备mol2格式文件。
   **Open Babel 2.3.0** ：一个用于化学结构文件类型相互转换的软件。
   **CHARMM36力场文件**  [CHARMM主页](http://mackerell.umaryland.edu/charmm_ff.shtml)  [CHARMM下载 ](http://mackerell.umaryland.edu/download.php?filename=CHARMM_ff_params_files/charmm36-jul2020.ff.tgz)
   Python 2.x 或 Python 3.x
   NetworkX 1.11 或 NetworkX 2.3
   转换文件
   [cgenff_charmm2gmx_py2_nx1.py](http://mackerell.umaryland.edu/download.php?filename=CHARMM_ff_params_files/cgenff_charmm2gmx_py2.py) Python 2.x, NetworkX 1.11
   [cgenff_charmm2gmx_py3_nx1.py](http://mackerell.umaryland.edu/download.php?filename=CHARMM_ff_params_files/cgenff_charmm2gmx_py3_nx1.py) Python 3.x, NetworkX 1.11
   [cgenff_charmm2gmx_py3_nx2.py] (http://mackerell.umaryland.edu/download.php?filename=CHARMM_ff_params_files/cgenff_charmm2gmx_py3_nx2.py) Python 3.x, NetworkX 2.3

使用方法

制作CGenFF可识别的.str文件

   - 制作.mol2格式的坐标文件，氢原子也包含在内。最好使用Open Babel软件进行转换。
   - 在CGenFF网站上传.mol2文件，此处有三个复选框：
      Guess bond orders from connectivity
      Include parameters that are already in CGenFF
      Use CGenFF legacy v1.0
    一般情况下都不用选。选项意义如字面所言。选项一重新判断键价，有一定概率出错，如果时用于其他软件的拓扑，不用选择。选项二会将原力场中已包含的信息一并列出，在后续格式转换的过程中可能会出现重复。

.str文件中会有一个penalty值，该值小于10表明结果较好，在10到50之间则说明需要进行一定的验证，大于50则需要更广泛的验证。

转换为GROMACS格式

     格式转换需要用到 [cgenff_charmm2gmx_py3_nx2.py] (http://mackerell.umaryland.edu/download.php?filename=CHARMM_ff_params_files/cgenff_charmm2gmx_py3_nx2.py)，同时需要预先下载好GROMACS可用的[CHARMM力场](http://mackerell.umaryland.edu/download.php?filename=CHARMM_ff_params_files/charmm36-jul2020.ff.tgz)
     程序运行命令为
     `python cgenff_charmm2gmx_py3_nx2.py RESNAME *.mol2 *.str charmm.ff`
     **RESNAME** 为残基名称，在 .str 文件中见“XXX 0.000!”字段中的XXX
     *.mol2 为 .mol2 文件
     *.str 为 .str 文件
     charmm36 为立场文件夹名称
